开发语言：Java

说明：
1.需要Java运行环境,首先要安装jdk，配置环境变量。
2.可以跨平台，window下执行rua.bat,mac下执行run.sh。
2.支持的excel文件格式：xls、xlsx。
3.支持在单元格中插入其他单元格内容。

excel配置表格式：
第一行为数据类型说明，第二行key，第三行为key描述。

使用说明：
1.打开config.cfg，配置相关信息。
2.安装好jdk后，可以双击run.bat（windows）和run.sh（Mac、Linux）两个脚本，也可以命令行输入java -jar exceltolua.jar。

注意：
执行后需要关注窗口里的输出，导出成功或失败会有提示！
